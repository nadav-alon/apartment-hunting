ברוך הבא למפת הדירות

שלב 1 מתוך 3 — איך מתחילים

  • לחץ + (או כפתור הפלוס הצף) להוספת דירה ידנית
  • לחץ על דירה ברשימה לעריכה ולהוספת הערות וסטטוס
  • עבור בין רשימה למפה בלחצנים בתחתית המסך
  • לחץ סינון לסנן לפי עיר, מחיר, סטטוס ועוד

שלב 2 מתוך 3 — ייבוא דירות מ-AI

הדרך המהירה להתחיל: שלח לכל AI (Claude, ChatGPT, Gemini) את הדירות שמצאת — הוא יחזיר JSON מוכן לייבוא.

  1. פתח את הקובץ import-prompt.md והעתק את תוכנו
  2. הדבק בשיחת AI ואחריו את פרטי הדירות (טקסט, קישורים, PDF...)
  3. שמור את תשובת ה-AI כקובץ .json
  4. לחץ ⋯ ← ייבוא ובחר את הקובץ (או הדבק ישירות)

שלב 3 מתוך 3 — סנכרון בין מכשירים

כדי שהנתונים יסתנכרנו בין הטלפון למחשב:

  1. צור טוקן ב-github.com ← Settings ← Developer settings ← Personal access tokens ← סמן scope: gist
  2. לחץ ⋯ ← סנכרון והדבק את הטוקן
  3. סרוק את קוד ה-QR שיוצג ממכשיר אחר — וזהו!

סנכרון בין מכשירים

GitHub Token (scope: gist)
סרוק ממכשיר נוסף
הקוד מכיל את הטוקן ו-Gist ID — אל תשתף עם אחרים
